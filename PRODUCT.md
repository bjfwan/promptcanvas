# Product

## Register

product

## Users

PromptCanvas is for image creators who want a local-first AI image studio in the browser. They write prompts, tune generation settings, attach reference images, review outputs, and reuse history without handing project data to an app backend.

## Product Purpose

PromptCanvas provides a pure frontend workspace for OpenAI-compatible image generation and edits. Success means users can move from a rough visual idea to generated images, variations, saved local history, and reusable prompt context with minimal friction.

## Brand Personality

Calm, precise, craft-minded. The interface should feel like an atelier workbench: focused, trustworthy, and quietly expressive without distracting from the canvas.

## Anti-references

Avoid marketing-page spectacle, ornamental SaaS gradients, heavy decorative glass effects, and noisy color use. Avoid anything that makes the generation workspace feel less reliable or less task-focused.

## Design Principles

Keep the canvas primary: surrounding UI should frame the work, not compete with it.

Use color semantically: reserve the teal action color for primary actions, active states, and feedback.

Respect local-first trust: communicate control, reversibility, and privacy through calm UI behavior.

Prefer dense clarity over decoration: tools should be scannable, consistent, and quick to operate.

Support creative iteration: history, references, settings, and prompt context should remain close at hand.

## Accessibility & Inclusion

Target WCAG AA contrast for text and controls. Preserve keyboard access, focus visibility, reduced-motion behavior, and color-independent state cues across the workspace.
